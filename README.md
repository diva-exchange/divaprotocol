# DIVA Protocol

Interface between DIVA Blockchain and Diva UI.

## Configuration
The configuration can be controlled using environment variables.

### IP
Default: 127.0.0.1

### PORT
Default: 19720

### URL_BLOCK_FEED
Default: ws://172.19.72.21:17469

### URL_API_CHAIN
Default: http://172.19.72.21:17468
